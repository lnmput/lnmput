问题

我们发现获取到的服务器信息不正确:
1、HTTP_HOST:localhost:8081
2、REMOTE_ADDR：127.0.0.1 (客户端的IP）
这会对我们的开发造成“迷茫”，因此要在nginx配置里加入
1、proxy_set_header Host $host;
2、proxy_set_header X-Real-IP $remote_addr;
