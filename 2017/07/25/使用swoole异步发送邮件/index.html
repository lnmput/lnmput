安装swoole

pecl 安装:
pecl install swoole

看命令行提示，如果它提示说没有写php.ini，则自己手动在PHP.ini后面加上：
extension = "swoole.so"

查看与关闭

swoole好像没有很便捷的关闭方式。所以只能直接通过关闭进程来关闭。
查看命令：
ps -ef | grep php

结束单个进程：
kill -9 {进程号}

结束所有进程的命令：
killall -9 php
